PAD Editor “ダブルチーズバーガー”

MKNPAD (0.03.02.ααα) since 2019-11-15, Double cheeseburger (0.001.200302.α) since 2020-03-02

PAD Editor (MKNPAD) は現在開発中です。 機能の一部は予告なしに変更される恐れがあります。 Internet Explorer と Microsoft Edge (EdgeHTML) では一部、もしくは全部の機能が利用できません。

ダブルチーズバーガー
  • 筑前煮との比較
    • 次のコマンドは廃止になりました: $, ADD, SUB, LET, MOV, MUL, DIV, MOD, INC, DEC AMD, XOR, OR, SHL, SHR, NOT, NEG, コナミコマンド, GO
    • 条件ブロックの比較コマンドは廃止されました。 代わりに疑似 C コードが利用できます。
    • コマンドブロックのコマンドがレガシコマンドに合致しない場合は疑似 C コードとして処理されます。
  • ダブルチーズバーガーの新機能
    • @ コマンドが追加されました。
